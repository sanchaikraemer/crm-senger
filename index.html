🔑 Chave da API Anthropic (para IA)

Necessária para usar o botão "🤖 Importar Print". Sua chave fica salva apenas no seu navegador (localStorage) — nunca é enviada para nenhum servidor nosso.

Como obter sua chave:
1. Acesse console.anthropic.com/settings/keys
2. Clique em "Create Key"
3. Copie a chave que começa com sk-ant-...
4. Cole abaixo e clique em Salvar
🔒 A chave é armazenada somente no seu navegador via localStorage.
Nunca é enviada ao GitHub nem a qualquer servidor externo — apenas diretamente à API da Anthropic na hora de analisar um print.
